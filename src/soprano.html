Sobre el Proyecto

¿Qué es Soprano?

Soprano: Mercados & Finanzas es una aplicación móvil completa para inversores y traders que necesitan seguir los mercados globales. La aplicación ofrece cotizaciones en tiempo real de acciones de NYSE, NASDAQ, índices internacionales y commodities, con una interfaz simple e intuitiva que facilita el análisis y la toma de decisiones.

¿Por qué nació este proyecto?

El mundo financiero es cada vez más globalizado y los inversores necesitan acceder a información de múltiples mercados simultáneamente. Muchas aplicaciones existentes son complejas, están sobrecargadas de funciones innecesarias o carecen de datos actualizados.

Soprano nació para llenar este vacío, ofreciendo una experiencia simplificada pero completa: datos en tiempo real, noticias financieras relevantes, listas de ganadores y perdedores del día, acciones en tendencia, y todo esto en una interfaz limpia que funciona en español, inglés y portugués, adaptándose a las preferencias de tema de cada usuario.

Stack Tecnológico

Frontend Mobile

React Native

Framework multiplataforma para desarrollo móvil nativo en iOS y Android, garantizando una experiencia fluida y rendimiento óptimo.

TypeScript

Tipado estático para JavaScript que proporciona mayor robustez, mantenibilidad y detección temprana de errores en el código.

Expo

Plataforma que acelera el desarrollo de React Native con herramientas preconstruidas y actualizaciones over-the-air.

React Query

Biblioteca para gestión de estado del servidor, manejo de caché y sincronización de datos en tiempo real, optimizando las peticiones API.

i18n

Internacionalización completa con soporte para español, inglés y portugués, permitiendo a usuarios de diferentes regiones disfrutar la app en su idioma nativo.

Soporte de Temas

Sistema de temas personalizables que permite a los usuarios elegir entre modo claro y oscuro, adaptándose a sus preferencias visuales.

Backend API

FastAPI

Framework moderno de Python para construir APIs RESTful de alto rendimiento con validación automática, documentación interactiva y tipado estático.

Arquitectura en Capas

Estructura modular con separación clara entre rutas, servicios, repositorios y modelos, garantizando escalabilidad y mantenibilidad.

Patrones de Diseño

Implementación de patrones como Repository, Dependency Injection y Factory, mejorando la estructura del código y facilitando el testing y mantenimiento.

Integración de APIs

Consumo de APIs financieras globales para obtener cotizaciones en tiempo real de NYSE, NASDAQ y mercados internacionales.
