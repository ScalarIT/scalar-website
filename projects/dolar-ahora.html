Sobre el Proyecto

¿Qué es Dólar ahora?

Dólar ahora es una aplicación móvil diseñada para el público argentino que necesita mantenerse informado sobre las cotizaciones de las diferentes variantes del dólar en Argentina. Con actualizaciones en tiempo real, la app proporciona información precisa y confiable sobre el dólar oficial, blue, MEP, CCL y más.

¿Por qué nació este proyecto?

En Argentina, el mercado cambiario es altamente volátil y complejo, con múltiples tipos de cambio que varían constantemente. Los usuarios necesitaban una herramienta simple, rápida y confiable para consultar estas cotizaciones al instante, sin tener que navegar por múltiples sitios web o lidiar con información desactualizada.

Dólar ahora surgió como respuesta a esta necesidad, ofreciendo una experiencia móvil optimizada con datos en tiempo real, un convertidor de divisas integrado y análisis del mercado argentino, todo en una interfaz intuitiva y fácil de usar.

Stack Tecnológico

Frontend Mobile

React Native

Framework multiplataforma para desarrollo móvil nativo en iOS y Android, garantizando una experiencia fluida y rendimiento óptimo.

TypeScript

Tipado estático para JavaScript que proporciona mayor robustez, mantenibilidad y detección temprana de errores en el código.

Expo

Plataforma que acelera el desarrollo de React Native con herramientas preconstruidas y actualizaciones over-the-air.

React Query

Biblioteca para gestión de estado del servidor, manejo de caché y sincronización de datos en tiempo real, optimizando las peticiones API.

Backend API

FastAPI

Framework moderno de Python para construir APIs RESTful de alto rendimiento con validación automática, documentación interactiva y tipado estático.

Arquitectura en Capas

Estructura modular con separación clara entre rutas, servicios, repositorios y modelos, garantizando escalabilidad y mantenibilidad.
